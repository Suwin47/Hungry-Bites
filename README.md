HungryBites 🍔🍕🍜

HungryBites is a modern and responsive food ordering frontend website developed using HTML, CSS, JavaScript, and Tailwind CSS.
The project is inspired by popular food delivery platforms like Swiggy and Zomato and provides a smooth user experience for browsing restaurants, offers, and food menus.

🚀 Features
Responsive Design for Mobile & Desktop
Modern UI/UX Design
Restaurant Listing Page
Restaurant Menu Pages
Search Functionality
Category Filter System
Food Offer Section
Checkout Page
Success Page
Hover Animations & Effects
Mobile Navigation Menu
Interactive Restaurant Cards
🛠️ Technologies Used
HTML5
CSS3
JavaScript
Tailwind CSS
